# PoC 3: 데이터 모니터링 Tool

## 목적 (과제 명세)

**현재 저장된 데이터 상태를 콘솔에서 실시간 조회**할 수 있는 관리자 도구.

## 완료 기준 (Definition of Done)

- [ ] PoC 2와 동일한 데이터 소스(JSON/DB)에 연결되는 독립 실행 CLI
- [ ] 전체 데이터 테이블 출력 (시료 / 주문)
- [ ] 조건 조회 (상태별 주문, 시료 ID/이름 검색)
- [ ] 요약 통계 (총 시료 수, 총 재고, 상태별 주문 건수)
- [ ] 실시간 반영: 새로고침(R) 시 디스크에서 재로드 → 외부 변경 즉시 확인 가능
- [ ] 데이터 소스 경로를 인자/설정으로 주입 (하드코딩 금지)
- [ ] README.md (연결 방법, 명령 목록, 화면 캡처/예시)

## 권장 메뉴 구성

```
[1] 시료 전체 조회   [2] 주문 전체 조회
[3] 상태별 주문 조회 [4] 요약 통계
[R] 새로고침         [0] 종료
```

## 구현 메모

- PoC 2의 Repository(읽기 메서드)를 재사용하면 일관성·재활용성 모두 충족
- 테이블 출력 시 컬럼 정렬(고정폭) — 한글 폭 주의
- 매 조회마다 재로드하는 방식이면 "실시간" 요건을 가장 단순하게 충족
- 읽기 전용 원칙. 시연 시나리오: 다른 프로세스(또는 Dummy 생성기)로 데이터 추가
  → 모니터에서 새로고침 → 반영 확인
