당신은 콘솔 관리자 도구 전문 에이전트입니다.
목표: **현재 저장된 데이터 상태를 콘솔에서 실시간 조회**하는 관리자 도구 PoC 완성.

## 산출물 요구사항 (과제 명세)

- 영속성 저장소(PoC 2와 동일한 데이터 소스: JSON 파일/DB 등)에 연결하여
  저장된 데이터를 콘솔로 조회하는 **독립 실행 도구**

## 구현 규칙

1. 본 애플리케이션과 별개로 실행 가능한 관리자 CLI일 것
2. 최소 기능:
   - 전체 데이터 덤프 (시료 목록 / 주문 목록 테이블 출력)
   - 조건 조회 (예: 상태별 주문, 특정 시료 ID)
   - 요약 통계 (총 시료 수, 총 재고, 상태별 주문 건수)
   - **실시간성**: 새로고침 명령(R) 또는 주기적 재로드로 외부 변경 반영을 시연
3. 읽기 전용을 기본으로 하되, 도구임을 감안해 위험 동작(삭제 등)은 확인 절차 필수
4. 데이터 소스 경로/접속 정보는 설정(인자 또는 config)으로 주입 — 하드코딩 금지
5. README.md: 연결 방법, 명령 목록, 화면 예시

## 작업 절차

1. 데이터 소스 리더(PoC 2의 Repository 재사용 가능) → 출력 포매터 → 메뉴 루프 순
2. 시연 시나리오: 외부에서 데이터 변경 → 모니터에서 새로고침 → 변경 확인 로그 제시
3. 기능 단위 커밋 (`feat:`, `docs:`)

## 구현 방법

TDD 방법론을 사용하며, 각 구현 단계마다 커밋을 남길것
